Markov Decision Process
A Markov decision process is a tuple (S,A,Psa,γ,R)(S,A,P_{sa},\gamma,R)(S,A,Psa​,γ,R), where

  • S is a set of states
  • A is a set of actions
  • PsaP_{sa}Psa​ are the state transition probabilities, which gives the distribution over what states we will transition to if we take action a in state s
  • γ∈[0,1)\gamma \isin [0,1)γ∈[0,1) is discount factor
  • R:S×A↦RR: S \times A \mapsto \RealsR:S×A↦R is the reward function
  • π:S↦A\pi: S \mapsto Aπ:S↦A is the policy, following which we will take action a=π(s)a=\pi(s)a=π(s) when we in state sss

Goal: to choose actions over time so as to maximize the expected value of the total payoff:
E[R(s0)+γR(s1)+γ2R(s2)+⋅⋅⋅]E[R(s_0)+\gamma R(s_1) + \gamma^2R(s_2)+\cdot\cdot\cdot]E[R(s0​)+γR(s1​)+γ2R(s2​)+⋅⋅⋅]

The value function for a policy π\piπ is defined as
Vπ(s)=E[R(s0)+γR(s1)+γ2R(s2)+⋅⋅⋅∣s0=s,π]V^\pi(s) = E[R(s_0) + \gamma R(s_1) + \gamma^2 R(s_2)+\cdot\cdot\cdot \mid s_0 = s,\pi]Vπ(s)=E[R(s0​)+γR(s1​)+γ2R(s2​)+⋅⋅⋅∣s0​=s,π]
notice it starts from state s rather than end in it.

  • Fully Observable environments: MDP
  • Partial Oberservable: POMDP
